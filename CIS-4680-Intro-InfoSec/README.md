# CIS 4680 – Introduction to Information Security and Privacy (Write-up)

**Institution:** Georgia State University

**Platform:** ACI Learning Skill Labs

**Total Labs Completed:** 14

Hands-on lab coursework covering foundational and intermediate cybersecurity concepts including network analysis, malware deployment, cryptography, incident response, and system hardening across Windows and Linux

## Skills Demonstrated

- **Network Security:** Wireless analysis, packet capture, protocol implementation, IDS/IPS
- **System Administration:** Active Directory, GPO, multi-OS backup strategies, patch management
- **Offensive Security:** RAT deployment, password cracking, Linux exploitation, vulnerability scanning
- **Defensive Security:** System hardening, AV configuration, security policy enforcement
- **Cryptography:** Public key infrastructure, GPG encryption, digital signatures
- **Incident Response:** Forensic analysis, evidence handling, log review, IR procedures
